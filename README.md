## Project setup

To run this project locally you need to install Flutter and its dependences with ``flutter pub get``

step 1:

```shell
git clone https://github.com/lebe24/Quix.git
``````
step 2:
````shell
amplify init
``````
step 3:
````shell
amplify push
``````
